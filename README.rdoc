== README

* Ruby version: 2.2.1

* System dependencies
	- Bundle 1.11.2

* How to run the project:
	in InLocoSelecao folder, run:
		bundle install
		bin/rails server -p 3000

* How to run the test suite
	in InLocoSelecao folder, run:
		bin/rails test

* Using devise for auth

* Using SendGrid and confirmable module for email confirmation
